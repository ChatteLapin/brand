# ブランド使用ガイドライン

ChatteLapin (= 個人事業屋号シャラパン) のブランド資産使用方針。

## ロゴ

- **メインロゴ**: `logo/chattelapin.png` (= 2048x2048 PNG)
- 改変禁止 (= 切り抜き / 色変更 / 反転 / 文字追加は本 repo 内で **派生 file** を作る形で対応)
- 商標性が確立した後は別途利用規約

## 使用許諾

- ChatteLapin org 配下の prj は自由に使用可 (= talent / business / volt / internal / brand)
- 外部利用 (= 取材・記事掲載等) はユーザー (= 金井大延) に都度連絡

## 派生 file 命名規約

| ファイル名 | 用途 |
|---|---|
| `chattelapin.png` | メイン (= 2048x2048) |
| `chattelapin-<size>.png` | resize (= 512 / 192 等) |
| `chattelapin-mono.png` | モノクロ版 (= 後で作成) |
| `chattelapin-white.png` | 反転版 / 白背景用 (= 後で作成) |
| `chattelapin.svg` | ベクター版 (= 後で作成) |

## 使用禁止事項

- ロゴの過度な変形 (= 縦横比率を崩す、 大幅にトリミングする)
